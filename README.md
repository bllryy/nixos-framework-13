# nixos-config

## Setup (existing NixOS install)

### 1. Enable flakes
```bash
echo 'experimental-features = nix-command flakes' | sudo tee -a /etc/nix/nix.conf
```

### 2. Clone and enter repo
```bash
git clone https://github.com/bllryy/nixos-framework-13.git ~/nixos-config
cd ~/nixos-config
```

### 3. Add your hardware config
```bash
nixos-generate-config --show-hardware-config > modules/hosts/framework/hardware-configuration.nix
```

Then replace `modules/hosts/framework/hardware.nix` contents with:
```nix
{ self, inputs, ... }: {
  flake.nixosModules.frameworkHardware = { ... }: {
    imports = [ ./hardware-configuration.nix ];
  };
}
```

### 4. Generate noctalia config
```bash
nix run nixpkgs#noctalia-shell ipc call state all > modules/features/noctalia.json
```

### 5. Edit configuration
- `modules/hosts/framework/configuration.nix` — timezone, extra packages
- `modules/features/niri.nix` — keybinds, startup apps, layout

### 6. Apply
```bash
sudo nixos-rebuild switch --flake .#framework
```

---

## Fresh install (from ISO)

```bash
# Partition and mount disks at /mnt, then:
git clone <your-repo>
cd nixos-config
bash install.sh
```

## File structure

```
.
├── flake.nix
├── install.sh
└── modules
    ├── hosts
    │   └── framework
    │       ├── default.nix               # nixosConfiguration entry point
    │       ├── configuration.nix         # packages, services, user
    │       ├── hardware.nix              # wraps hardware-configuration.nix
    │       └── hardware-configuration.nix  # generated, gitignore or commit
    └── features
        ├── niri.nix                      # niri compositor + keybinds
        ├── noctalia.nix                  # noctalia bar/launcher
        └── noctalia.json                 # generated by noctalia ipc call
```

## Keybinds (niri)

| Key | Action |
|-----|--------|
| `Mod+Return` | Open Ghostty |
| `Mod+Q` | Close window |
| `Mod+S` | Toggle Noctalia launcher |
